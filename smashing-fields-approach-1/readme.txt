== Description ==
Firefly Plugin has 2 features.

1. User can choose to enable shortcodes in widgets.
2. User can choose background color for a variety of DOM elements.

== Installation ==

1. Upload "firefly-plugin" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
