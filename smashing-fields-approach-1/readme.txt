== Description ==

Firefly Demo Plugin has only 1 feature: the user can choose to enable shortcodes in widgets.

== Installation ==

1. Upload "firefly-plugin" to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.

== Frequently Asked Questions ==

= What is this plugin really for? =
This plugin was developed as a teaching tool. It is intended to serve as a template for anyone seeking to get started with writing WordPress plugins.
